# InMobi

1、在 GIO 广告监测下创建监测链接，渠道选择为“Inmobi”，生成监测链接。

2、进入 Inmobi 后台，创建主张，选择 GrowingIO 渠道，创建成功后，将 Inmobi 生成的 ID 复制并填写到 propertyID 当中。
