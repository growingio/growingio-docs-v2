# 访问事件及变量

## 访问事件的定义

在GrowingIO的数据模型中，访问事件代表：

* Web端：用户在一定时间内首次打开网站页面
* 移动端：用户在一定时间内首次打开移动应用
* 小程序：用户在微信中打开小程序

一次访问会生成一个session值，各平台的session刷新时间如下：

* Web端：用户无操作30分钟后session值过期，30分钟后的操作生成新的session值。
* 移动端：App退出30秒后再进入，刷新session值。
* 小程序：小程序关闭5分钟后再进入，刷新session值。
