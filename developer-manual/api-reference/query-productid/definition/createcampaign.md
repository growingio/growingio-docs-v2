# 新建推广活动

### URL

https://www.growingio.com/api/v1/projects/_**project\_uid**_/meta/campaigns

### 请求类型

POST

### 请求头参数

请参考[认证](../../authenticate/)&gt;[公共请求参数](../../authenticate/head-parameter.md)获取。
