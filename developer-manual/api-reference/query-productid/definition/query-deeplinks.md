# 查询监测链接（吸引用户直接打开App）

### URL

https://www.growingio.com/api/v1/projects/_**project\_uid**_/meta/deeplinks

### 请求类型

GET

### 请求头参数

请参考[认证](../../authenticate/)&gt;[公共请求参数](../../authenticate/head-parameter.md)获取。

### 参数说明与示例

| :--- | :--- | :--- | :--- |
| project\_uid | string | 是 | 项目UID。 |
