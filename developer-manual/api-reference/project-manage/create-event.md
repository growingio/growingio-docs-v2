# 创建埋点事件

### URL

https://www.growingio.com/v1/api/projects/{project\_uid}/dim/events

### 请求类型

POST

### 请求头参数

请参考[认证](../authenticate/)&gt;[公共请求参数](../authenticate/head-parameter.md)获取。
