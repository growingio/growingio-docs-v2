# 获取留存分析数据

### URL

https://www.growingio.com/v2/projects/{project\_uid}/retentions/{retention\_id}.json

### 请求类型

GET

### 请求头参数

公共头部请参考[公共请求头参数](../../authenticate.md)。

### 参数说明与示例

| :--- | :--- | :--- | :--- |

| project\_uid | string | 是 | 项目UID。 |
