# 获取事件分析数据

### URL

https://www.growingio.com/v2/projects/{project\_uid}/charts/{chart\_id}.json

### 请求类型

GET

### 请求头参数

请参考[认证](../../authenticate/)&gt;[公共请求参数](../../authenticate/head-parameter.md)获取。
