# 运行时API

GrowingIO为App提供运行时随意调用的API，使用方法如下：

```java
// 得到 GrowingIO 实例后可以调用其中 API
GrowingIO gio = GrowingIO.getInstance();
gio.setUserId("张溪梦");
```

{% hint style="danger" %}
GrowingIO所有API都需要在主线程调用。 
{% endhint %}
